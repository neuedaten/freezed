Project structure

A Freezed project is just a handful of folders:

my-site/
├─ content/        # your pages, grouped by content type
│  └─ pages/
│     └─ home/
│        ├─ index.html      # a Fluid template
│        └─ variables.php   # variables for this page
├─ themes/         # one or more stackable themes
├─ static/         # files copied verbatim into the build
├─ public/         # generated output (git-ignored)
└─ freezed.config.php

The resource ViewHelper

Reference a theme asset and Freezed will copy it into the build and return its public path: 

<link rel="stylesheet"
  href="{freezed:resource(path: 'assets/css/main.css', context: 'theme')}">
