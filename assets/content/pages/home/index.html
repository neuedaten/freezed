Components are reusable, typed template tags with slots. They live in the theme's templates/components/ folder and are called like <component:callout> — no f:render needed.

Templates, not magic

Freezed renders every page with the TYPO3 Fluid template engine — the same engine that powers TYPO3 CMS. You get layouts, partials, sections and ViewHelpers, compiled to plain static HTML.

This whole page is a Fluid template in content/pages/home/index.html. Its variables live next to it in variables.php, and the look comes from the 00_default theme. Edit any of them and run freezed build to regenerate the site.
