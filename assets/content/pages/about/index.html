Freezed is a small static site generator with one defining trait: it renders your pages with the TYPO3 Fluid template engine.

Why Fluid?

Fluid is a mature, well-documented templating language with a clean separation of layouts, partials and sections. If you have worked with TYPO3, it will feel immediately familiar. If you haven't, it is an approachable, HTML-first syntax with first-class support for reusable components and custom ViewHelpers.

How a build works

Running freezed build does three things:

  1. It scans content/ for content types and pages.
  2. It renders each page through the themes in themes/.
  3. It writes static HTML and copied assets into public/.
No databases, no runtime, no surprises — just templates in, static files out.

Status

Freezed is currently in beta. The core build pipeline is stable, and the API may still evolve before a 1.0 release. Feedback and issues are very welcome on GitHub.
